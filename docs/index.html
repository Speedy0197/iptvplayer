📖 Installation Instructions
  1. Tap "Open TestFlight" button above
  2. Accept the invite in TestFlight app
  3. Tap "Install" and wait for the app to download
  4. Launch StreamPilot from your home screen
📖 Installation Instructions
  1. Download the DMG file and double-click it
  2. Drag StreamPilot into the Applications folder
  3. Open Applications, then right-click StreamPilot and choose "Open" the first time
  4. If macOS blocks the app, go to System Settings → Privacy & Security and click "Open Anyway"
